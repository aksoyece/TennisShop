# 🎾 Aksoy TennisShop

A tennis equipment e-commerce website built with HTML, CSS, and JavaScript.

## Features

- Product listing with category filtering (Rackets, Balls, Shoes, Accessories, Clothing)
- Shopping cart with add/remove functionality
- Gender-based clothing filter (Men/Women)
- Responsive design
- Product images with consistent styling

## Pages

- **index.html** — Homepage with featured products and hero section
- **products.html** — Full product listing with category filters and shopping cart

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Or use Live Server extension in VS Code
